# PVM v0.1.0

Windows 上的 PHP 多版本管理工具（MVP）。

## 下载

64 位 Windows 选 x64，32 位选 x86：


| 类型      | x64                          | x86                          |
| ------- | ---------------------------- | ---------------------------- |
| MSI（推荐） | `pvm-v0.1.0-windows-x64.msi` | `pvm-v0.1.0-windows-x86.msi` |
| ZIP     | `pvm-v0.1.0-windows-x64.zip` | `pvm-v0.1.0-windows-x86.zip` |
| exe     | `pvm-v0.1.0-windows-x64.exe` | `pvm-v0.1.0-windows-x86.exe` |


SHA256 见附件 `checksums.txt`。

## 快速开始

```powershell
# 安装 MSI 或运行 ZIP 内 install.ps1 后，新开终端
pvm install 8.3
php -v
```

## 系统要求

- Windows 10 / 11
- 包架构需与系统一致（x64 / x86）
