# PVM v0.1.0

首个 MVP 发布：在 Windows 上安装、切换和管理多个 PHP 版本。

## 下载

请按 **Windows 系统位数** 选择对应架构（64 位系统选 x64，32 位系统选 x86）：

| 类型 | x64（64 位 Windows） | x86（32 位 Windows） |
|------|----------------------|------------------------|
| MSI（推荐） | `pvm-v0.1.0-windows-x64.msi` | `pvm-v0.1.0-windows-x86.msi` |
| ZIP 便携包 | `pvm-v0.1.0-windows-x64.zip` | `pvm-v0.1.0-windows-x86.zip` |
| 单文件 exe | `pvm-v0.1.0-windows-x64.exe` | `pvm-v0.1.0-windows-x86.exe` |

所有产物的 SHA256 见同页附件 **`checksums.txt`**。

### 安装示例

**MSI：**

```powershell
msiexec /i pvm-v0.1.0-windows-x64.msi
```

**ZIP：**

```powershell
Expand-Archive pvm-v0.1.0-windows-x64.zip -DestinationPath .
cd pvm-v0.1.0-windows-x64
.\install.ps1
```

安装后 **新开终端** 运行 `pvm version`，然后：

```powershell
pvm install 8.3
php -v
```

## 快速开始

```powershell
# 一条命令安装并激活（默认 auto_use_after_install = true）
pvm install 8.3

# 当前终端即可 php -v，无需单独 setup
php -v
pvm list

# 切换版本
pvm use 8.2
php -v
```

## 本版本功能

- **版本管理**：`install`、`uninstall`、`use`、`list`
- **远程索引**：`list-remote` / `ls-remote`（表格展示）
- **别名与默认**：`alias`、`default`
- **环境**：`setup`、`unsetup`、`refresh`、`path`
- **离线安装**：`install --from-zip`
- **SHA256** 校验官方 PHP 包

## 环境配置（v0.1.0 简化）

| 能力 | 说明 |
|------|------|
| 首次 `use` 自动 setup | PATH 无 `current` 时自动写入用户环境 |
| `use` 后会话刷新 | 当前终端立刻可用 `php -v` |
| MSI / ZIP 预置 PATH | 安装包写入 `%USERPROFILE%\.pvm\current` 与 `PHP_HOME` |
| `auto_use_after_install` | 默认 `true`，`install` 后自动 `use` |

手动 `pvm setup` 仍可用；其它已打开终端请用 `pvm refresh`。

## 系统要求

- Windows 10 / 11
- x64 系统 → x64 包；x86 系统 → x86 包

## 已知限制

- 在线下载受网络影响；可用 `--from-zip`
- `-v` 日志、HTTP 代理尚未接入
- `config` / 扩展 / `.pvmrc` 计划在 v1.0
